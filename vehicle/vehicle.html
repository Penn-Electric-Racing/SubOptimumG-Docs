vehicle#

car_weight_distributor(self, backwards_pass=False)#

Updates the normal forces on each tire by computing and applying longitudinal and lateral weight transfer effects.

Parameters:

backwards_pass (bool) – Flag to indicate if the calculation is for a backwards pass, which inverts the longitudinal acceleration direction.

Default backwards_pass:

False

Returns:

(fl, fr, bl, br) - A tuple of normal forces (in Newtons) on each tire:

  • fl - front left tire

  • fr - front right tire

  • bl - back left tire

  • br - back right tire

More positive means more pushed into the ground.

Return type:

tuple[float, float, float, float]

Note

The calculation takes into account:

  • Vehicle mass, weight distribution, CG height

  • Wheelbase, front and rear track widths

  • Suspension behavior

  • Current longitudinal acceleration self.acc, current lateral acceleration self.lat_acc.

Examples

# Calculate normal forces during forward acceleration
fl, fr, bl, br = car.car_weight_distributor(backwards_pass=False)

# Calculate normal forces during braking
fl, fr, bl, br = car.car_weight_distributor(backwards_pass=True)

See also

longitudinal_weight_transfer(), lateral_weight_transfer()
